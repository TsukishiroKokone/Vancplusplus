# Van Lexicon Variables

> 本文档列出 Van Lexicon C++ 版内置变量。变量支持多轮嵌套解析，默认最多 52 轮。

## 嵌套规则

```text
[重复.[随机数字].🌸]
[计算.[长度.abc].*2]
[随机选择.猫猫|兔兔|[name]]
```

## 变量总览

当前版本按“变量族 = 一个可直接使用的变量名 / 参数化变量模板”统计，已提供 **520+ 个不同变量族**，不是只靠 `[数字N]` 这类模板膨胀实例数。非 HTTP 请求变量目标解析延迟为 5ms 以内；引擎会在无变量字符或无变化时提前停止，避免 52 层嵌套空转。

## 520+ 固定实用变量族

固定变量族由 `src/vanbot/variable_engine.hpp` 中的哈希表按 token 一次扫描解析，避免 520+ 次全量字符串替换。覆盖类别包括：

| 类别 | 约数量 | 示例 |
|---|---:|---|
| 布尔/开关/权限/符号 | 30+ | `[布尔真]`、`[开启]`、`[空格]`、`[左中括号]` |
| HTTP 状态短语与状态码 | 120+ | `[HTTP200]` → `OK`、`[HTTP码404]` → `404` |
| MIME 与 Content-Type 头 | 60+ | `[MIMEJSON]`、`[头JSON]` |
| HTTP 方法/日志级别/运行状态 | 40+ | `[方法POST]`、`[级别debug]`、`[状态在线]` |
| 单位/常用端口 | 70+ | `[单位MB]`、`[端口8080]` |
| 颜色/可爱符号/问候短语 | 90+ | `[颜色樱花HEX]`、`[猫爪]`、`[好耶]` |
| 日期方向/时区/货币 | 80+ | `[星期一]`、`[时区上海]`、`[货币人民币]` |
| 语言/国家/省份/星座/生肖 | 110+ | `[语言日语]`、`[国家日本]`、`[省份广东]` |
| 稀有度/天气/文件扩展/协议/配置键 | 90+ | `[稀有度SSR]`、`[天气晴]`、`[配置键sqlite_path]` |

## 少量参数化工具族

这些是模板族，额外提供大量实用写法，但不作为“520+ 固定变量族”的主要计数依据：

| 变量族 | 说明 | 示例 |
|---|---|---|
| `[数字N]` | 输出数字本身 | `[数字520]` → `520` |
| `[两位N]` | 小于 10 时补零 | `[两位7]` → `07` |
| `[百分N]` | 百分比文本 | `[百分80]` → `80%` |
| `[HTTPN]` | HTTP 状态码数字文本 | `[HTTP404]` → `404` |
| `[端口N]` | 端口号文本，适合拼配置 | `[端口8080]` → `8080` |
| `[KBN]` / `[MBN]` / `[GBN]` | KiB/MiB/GiB 转字节 | `[MB2]` → `2097152` |

## 统计、身份、格式化与控制变量

原有统计、身份、文本、计算、控制变量仍完整保留：`[收消息数]`、`[主人列表]`、`[艾特]`、`[重复.N.文本]`、`[长度.文本]`、`[计算.表达式]`、`[包含.文本.片段]`、`[url编码.文本]`、`[json转义.文本]`、`[CQ转义.文本]`、`[get.URL]`、`{条件}`、`(!错误!)`、`(-秒-)` 等均可继续使用。

## 注意事项

- 文本类截取和长度以 UTF-8 字节为单位，不按 Unicode 字符宽度计算。
- 参数变量内部若含 `]`，请使用 `[右中括号]` 等常量变量辅助嵌套。
- 嵌套变量会多轮解析，当前最大 52 层；复杂组合建议控制长度，避免无限展开。
- 520+ 固定变量族使用一次 token 扫描 + `unordered_map` 查询，不会循环枚举 520 个 `replace_all`。
- 5ms 是非 HTTP、非磁盘慢查询场景的目标值；实际速度会受回复长度、嵌套层数、CPU、存储后端影响。
