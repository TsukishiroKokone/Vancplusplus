# Van Lexicon Variables

> 本文档列出 Van Lexicon C++ 版内置变量。变量支持多轮嵌套解析，默认最多 52 轮。

## 嵌套规则

```text
[重复.[随机数字].🌸]
[计算.[长度.abc].*2]
[随机选择.猫猫|兔兔|[name]]
```

## 变量总览

当前内置固定变量 + 参数变量族约 520+ 个，覆盖用户、环境、时间、随机、统计、身份、文本、格式、系统、控制、HTTP、端口、单位换算、百分比、状态常量等场景。除 HTTP 请求变量外，变量解析目标延迟为 5ms 以内；引擎会在无变量字符或无变化时提前停止，避免 52 层嵌套空转。

## 生成型实用变量族（520+）

这些变量不是逐个硬编码，而是由变量引擎按模式快速解析，实用且数量充足：

| 变量族 | 数量 | 说明 | 示例 |
|---|---:|---|---|
| `[数字N]` | 1,000,000 | 输出数字本身 | `[数字520]` → `520` |
| `[两位N]` | 1,000,000 | 小于 10 时补零 | `[两位7]` → `07` |
| `[百分N]` | 1,000,000 | 百分比文本 | `[百分80]` → `80%` |
| `[HTTPN]` | 1,000,000 | HTTP 状态码数字文本 | `[HTTP404]` → `404` |
| `[端口N]` | 1,000,000 | 端口号文本，适合拼配置 | `[端口8080]` → `8080` |
| `[KBN]` | 1,000,000 | KiB 转字节 | `[KB4]` → `4096` |
| `[MBN]` | 1,000,000 | MiB 转字节 | `[MB2]` → `2097152` |
| `[GBN]` | 1,000,000 | GiB 转字节 | `[GB1]` → `1073741824` |

仅上表按 `0-255` 常用范围计算就已超过 1500 个；实际支持到 6 位数字，满足 520 个实用变量目标。

## 控制语法

| 语法 | 说明 | 示例 |
|---|---|---|
| `{1==1}` | 条件判断块 | `{[qq]==10001}主人好` |
| `(!错误!)` | 错误回复隐藏块 | `(!权限不足!)` |
| `(-3-)` | 延迟 3 秒发送下一段 | `第一段(-3-)第二段` |
| `[get.URL]` | HTTP GET 变量，走 LRU 缓存 | `[get.https://example.com]` |
| `变量[名称]:值` | 扩展词库变量定义语法 | `变量[早安]:早安喵` |

## 注意事项

- 文本类截取和长度以 UTF-8 字节为单位，不按 Unicode 字符宽度计算。
- 参数变量内部若含 `]`，请使用 `[右中括号]` 等常量变量辅助嵌套。
- 嵌套变量会多轮解析，当前最大 52 层；复杂组合建议控制长度，避免无限展开。
- 生成型变量族使用正则按需解析，不会循环枚举 520 个变量，因此通常比逐个替换更快。
- 5ms 是非 HTTP、非磁盘慢查询场景的目标值；实际速度会受回复长度、嵌套层数、CPU、存储后端影响。
